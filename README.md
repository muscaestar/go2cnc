Go to Cloud Native Computing!

## 云原生
- 是一种思想也是一套能力，由一系列技术工具来支持这个能力
- 全自动
- 动态环境（有能力自动解决单点故障）
- 依托容器技术
- 服务网格
- 不可变技术架构
- 声明式API
